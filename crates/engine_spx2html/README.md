# The `tectonic_engine_spx2html` crate

[![](http://meritbadge.herokuapp.com/jxoesneon-tectonic-engine_spx2html)](https://crates.io/crates/jxoesneon-tectonic-engine_spx2html)

> [!NOTE]
> This crate is part of the **FerroTeX** project, a specialized fork of Tectonic.
> It is published to crates.io as `jxoesneon-tectonic-engine_spx2html`.

This crate is part of [the Tectonic
project](https://tectonic-typesetting.github.io/en-US/). It provides and engine
that converts "SPX" (semantically paginated XDV) files, a customized Tectonic
output, to HTML.

- [API documentation](https://docs.rs/tectonic_engine_spx2html/).
- [Main Git repository](https://github.com/jxoesneon/tectonic/).
